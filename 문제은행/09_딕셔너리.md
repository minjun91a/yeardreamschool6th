## 문제
딕셔너리 `person = {"name": "민준", "age": 20}` 에서 이름(`name`) 값을 출력하세요.

## 풀이
```python
person = {"name": "민준", "age": 20}
print(person["name"])
```

## 문제
딕셔너리 `scores = {"국어": 90, "영어": 85, "수학": 100}` 의 과목과 점수를 한 줄에 하나씩 출력하세요.
(힌트: `.items()` 로 (이름표, 값) 쌍을 하나씩 꺼냅니다)

## 풀이
```python
scores = {"국어": 90, "영어": 85, "수학": 100}
for 과목, 점수 in scores.items():
    print(과목, 점수)
```

## 문제
딕셔너리 `cart = {"사과": 2}` 에 `"바나나"` 를 `3` 개 추가한 뒤, 딕셔너리 전체를 출력하세요.

## 풀이
```python
cart = {"사과": 2}
cart["바나나"] = 3
print(cart)
```

## 문제
딕셔너리 `person = {"name": "수아", "age": 19, "city": "서울"}` 에 들어 있는 **이름표(key)의 개수**를 출력하세요.
(힌트: `len()`)

## 풀이
```python
person = {"name": "수아", "age": 19, "city": "서울"}
print(len(person))
```
