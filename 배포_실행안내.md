# 학습올인원 서버 — Render 무료 배포 가이드

이 폴더를 그대로 GitHub에 올리고 Render에 연결하면 무료로 공개됩니다.
(운영자 키·개인 노트·진행상황은 들어있지 않습니다. 진행상황은 사용자 브라우저에, API 키는 사용자 본인 것(BYOK).)

## 준비물 (둘 다 무료)
- GitHub 계정
- Render 계정  →  https://render.com  (GitHub 로 가입하면 편함)

## 1) GitHub 레포에 올리기
이 폴더(학습올인원_서버배포)에서:
```bash
git init
git add .
git commit -m "학습올인원 서버 배포본"
git branch -M main
git remote add origin https://github.com/<당신아이디>/<레포이름>.git
git push -u origin main
```

## 2) Render에서 배포 (둘 중 하나)
### A. 블루프린트 (원클릭 — render.yaml 자동 인식, 추천)
Render 대시보드 → **New → Blueprint** → 위 레포 선택 → **Apply**.
`render.yaml` 이 빌드·실행·무료플랜을 자동 설정합니다.

### B. 수동 (블루프린트가 안 될 때)
**New → Web Service** → 레포 선택 → 아래 값 입력:
- Runtime: **Python**
- Plan: **Free**
- Build Command:  `pip install -r requirements.txt`
- Start Command:  `cd server && uvicorn app:app --host 0.0.0.0 --port $PORT`
- Health Check Path:  `/api/health`
→ **Create Web Service**.

## 3) 끝 — 사용
몇 분 뒤 `https://<서비스이름>.onrender.com` 주소가 발급됩니다(자동 HTTPS).
접속 → **설정 ⚙️** 에서 본인 API 키 등록 → 노트 변환·코딩 연습·AI 도우미·도움말 사용.

## 무료 티어 알아두기
- 15분간 방문이 없으면 잠들고, 다음 방문 때 자동으로 깨어납니다(첫 요청 약 1분, 이후 빠름). 꺼지는 게 아님.
- 월 750 인스턴스-시간 무료(서비스 1개 24시간 가동에 충분, 잠들면 더 적게 씀). 내 PC 와 무관하게 클라우드에서 구동.
- 콜드스타트가 거슬리면 UptimeRobot 등으로 10분마다 핑(선택, 무료).
- 한국 지연을 줄이려면 Render 에서 Region 을 Singapore 로 바꿔도 됩니다(선택). render.yaml 의 region 주석을 풀어도 됨.

## 업데이트
코드를 고치고 `git push` 하면 Render 가 자동으로 다시 배포합니다.

## 안전(이미 처리됨)
- 이 폴더엔 운영자 키·개인 노트·진행상황·저작 도구가 **없습니다**(생성기가 누출검사 완료).
- 서버는 사용자 키를 저장하지 않고(BYOK), 사용자 코드는 브라우저(Pyodide)에서만 실행합니다.
- 업로드는 3MB 제한. 로그는 화면(stdout)과 data/app.log 에 남습니다(쓰기 권한 필요).
